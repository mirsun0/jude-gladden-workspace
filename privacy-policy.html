Information Collection We only collect mobile phone numbers and text message content that you explicitly and voluntarily send to our system for private note-taking, processing, and reminder synchronization. ### 2. Absolute Non-Sharing of Mobile Information We recognize the importance of your privacy. Mobile information, phone numbers, or text message data collected for the purpose of SMS verification, note storage, or text messaging workflows will absolutely never be shared, sold, or rented to any third parties, affiliates, or partners for marketing or promotional purposes under any circumstances. ### 3. Data Storage & Security All incoming text messages are processed securely and stored on private, local server architecture. We implement strict digital security protocols to protect your text data from unauthorized access. ### 4. Message Frequency Message frequency varies based on user interaction. You will only receive text messages from this system in direct response to an action you take, such as a requested text-back notification, task reminder, or service confirmation. ### 5. Changes to This Policy We reserve the right to update this policy at any time. Any changes will be posted directly to this public page.
